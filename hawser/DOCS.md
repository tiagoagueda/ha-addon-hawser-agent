# Hawser Agent – Home Assistant

![Dockhand Logo](https://raw.githubusercontent.com/tiagoagueda/ha-addon-hawser-agent/main/hawser/logo.png)

Docker agent that connects Home Assistant to Dockhand.

## How it works

- Outbound connection to your Dockhand server (Edge mode)
- Requires Docker socket access

## Requirements

After installing, go to **Settings → Add-ons → Hawser Agent → Info** and disable **Protection Mode**.

## Security

⚠️ Docker socket access grants full control over the host system.
